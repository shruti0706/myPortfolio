# Shruti Agrawal - Portfolio Website

This is a personal portfolio website of Shruti Agrawal built with React to showcase my experience, projects, skills, and certifications.

## Features

- Responsive design optimized for all devices
- Interactive UI with smooth scrolling
- Project showcase section
- Skills and experience timeline
- Contact form
- Deployed with GitHub Pages

## Technologies Used

- React.js
- CSS3
- JavaScript
- GitHub Actions for CI/CD
- GitHub Pages for hosting

## Local Development

1. Clone the repository
 

2. Install dependencies
   ```
   npm install
   ```

3. Start the development server
   ```
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Deployment

This portfolio is automatically deployed to GitHub Pages using GitHub Actions whenever changes are pushed to the master branch.
